# LLMs.txt

We support [LLMs.txt](https://llmstxt.org/) files to make Koobiq React documentation easier to use with AI tools.

## Available resources

- [`llms.txt`](https://react.koobiq.io/llms.txt) — the main entry point for Koobiq React documentation.
- Detailed component and guide files linked from `llms.txt`.

## Usage with AI tools

Use [`llms.txt`](https://react.koobiq.io/llms.txt) as the starting point.

Support depends on the AI tool. Some tools can use these files directly, while others may require you to open or attach them manually.
