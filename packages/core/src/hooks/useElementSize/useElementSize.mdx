# useElementSize

<Status variant="experimental" />

A hook listens for size changes of the particular element and returns its width and height.

```tsx
import { useElementSize } from '@koobiq/react-core';
```

## Definition

```ts
export function useElementSize<T extends HTMLElement = any>(
  options?: ResizeObserverOptions
);
```
