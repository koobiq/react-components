# Calendar

<Status variant="experimental" />

Provides the behavior and accessibility implementation for a calendar component.
A calendar displays one or more date grids and allows users to select a single date.

```tsx
import { Calendar } from '@koobiq/react-components';
```

## Disabled

The `isDisabled` boolean prop makes the Calendar disabled. Cells cannot be focused or selected.

## Min and Max values

By default, Calendar allows selecting any date. The `minValue` and `maxValue` props can also be used to prevent
the user from selecting dates outside a certain range.
