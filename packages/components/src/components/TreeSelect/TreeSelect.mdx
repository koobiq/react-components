# TreeSelect

<Status variant="experimental" />

Select with hierarchical tree data.

## Import

```tsx
import { TreeSelect } from '@koobiq/react-components';
```

## Usage

<Story of={Stories.Base} />

## Props

<Props of={Stories.Base} />

## Content

TreeSelect accepts the same collection shape as `Tree`: either static
`TreeSelect.Item` children or `items` with a render function.

### Static collection

Use `TreeSelect.Item` when the tree structure is known in advance.

<Story of={Stories.StaticCollection} />

### Dynamic collection

Use `items` and `Collection` when the tree data comes from an external source or
changes over time.

<Story of={Stories.DynamicCollection} />

## Label

Use `label` when the field should be presented as a regular form control.

<Story of={Stories.WithLabel} />

Use `selectedKey` and `onSelectionChange` to control the selected node from the
