# IconItem

<Status variant="experimental" />

IconItem is a small colored container for an icon.
Use it to show a product, status, or object icon with a clear visual accent.

## Import

```tsx
import { IconItem } from '@koobiq/react-components';
```

## CSS Variables

Use CSS variables to customize the icon item.

```css
:root {
  --kbq-icon-item-size: var(--kbq-size-3xl);
  --kbq-icon-item-border-radius: var(--kbq-size-l);
  --kbq-icon-item-color: var(--kbq-icon-white);
  --kbq-icon-item-bg-color: var(--kbq-background-theme);
}
```
