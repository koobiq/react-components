# RadioGroup

<Status variant="experimental" />

A radio group allows a user to select a single item from a list of mutually exclusive options.

```tsx
import { RadioGroup, Radio } from '@koobiq/react-components';
```

<Story of={Stories.Base} />

## Props

<Props />

## Size

For `normal` or `big` radios, use the `size` props.

<Story of={Stories.Size} />

## Invalid

The `isInvalid` prop toggles the error state.

<Story of={Stories.Invalid} />

## Disabled

A radio group can be disabled by passing the `isDisabled` prop.
To disable an individual radio, pass `isDisabled` prop to the Radio instead.

<Story of={Stories.Disabled} />

## Label placement and alignment

Use `labelPlacement` and `labelAlign` props to set the position of the label.

See [Create a form](?path=/docs/components-input--docs#create-a-form) story for a two column form with a shared label column.

<Story of={Stories.LabelPlacementAlignment} />

## Value

### Default value

An initial, uncontrolled value can be provided to the `RadioGroup` using the `defaultValue` prop,
which accepts a value corresponding with the `value` property of each `Radio`.

<Story of={Stories.DefaultValue} />

### Controlled value

A controlled value can be provided using the `value` prop,
which accepts a value corresponding with the `value` prop of each `Radio`.
The `onChange` event is fired when the user selects a radio.

<Story of={Stories.ControlledValue} />

## Caption

The `caption` can be used to associate additional help text with a radio group.

<Story of={Stories.Caption} />

## Orientation

RadioGroups are vertically oriented by default. The `orientation` can be used to change the orientation to `horizontal`.
