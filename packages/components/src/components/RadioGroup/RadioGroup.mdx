# RadioGroup

<Status variant="experimental" />

A radio group allows a user to select a single item from a list of mutually exclusive options.

```tsx
import { RadioGroup, Radio } from '@koobiq/react-components';
```

<Story of={Stories.Base} />

## Props

<Props />

## Size

For `normal` or `big` radios, use the `size` prop.

<Story of={Stories.Size} />

## Error

The `error` property toggles the error state.

<Story of={Stories.Error} />

## Disabled

A radio group can be disabled by passing the `disabled` property.
To disable an individual radio, pass `disabled` to the Radio instead.

<Story of={Stories.Disabled} />

## Value

### Default value

An initial, uncontrolled value can be provided to the `RadioGroup` using the `defaultValue` property,
which accepts a value corresponding with the `value` property of each `Radio`.

<Story of={Stories.DefaultValue} />

### Controlled value

A controlled value can be provided using the `value` property,
which accepts a value corresponding with the `value` property of each `Radio`. The `onChange` event is fired when the user selects a radio.

<Story of={Stories.ControlledValue} />

## Description

The `description` can be used to associate additional help text with a radio group.

<Story of={Stories.Description} />

## Orientation

RadioGroups are vertically oriented by default. The `orientation` can be used to change the orientation to horizontal.

<Story of={Stories.Orientation} />
