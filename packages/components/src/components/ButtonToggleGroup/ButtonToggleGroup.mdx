# ButtonToggleGroup

<Status variant="experimental" />

Display multiple options and allow users to select a single option.

```tsx
import { ButtonToggleGroup, ButtonToggle } from '@koobiq/react-components';
```

## Disabled

### Group

All buttons within a ButtonToggleGroup can be disabled using the `isDisabled` prop.

<Story of={Stories.DisabledGroup} />

### Item

Individual items can be disabled using the `isDisabled` prop on each ButtonToggle.

## Long Text

If the content of a `ButtonToggle` is too wide for its container,
the text will be truncated with an ellipsis, and a tooltip showing
the full text will appear on hover or focus.

<Story of={Stories.LongText} />

## Selection

Setting a selected option can be done by using the `defaultSelectedKey` or `selectedKey` prop.
The selected key corresponds to the `id` prop of `ButtonToggle`.
