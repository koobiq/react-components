# Toggle

<Status variant="experimental" />

Toggle is a component that is used to quickly switch between two possible states: "on" and "off".

```tsx
import { Toggle } from '@koobiq/react-components';
```

## Value

### Default value

Toggles are not checked by default. The `defaultSelected` prop can be used to set the default state.

<Story of={Stories.DefaultValue} />

### Controlled value

The `isSelected` prop can be used to make the checked state controlled.
The `onChange` event is fired when the user presses the toggle, and receives the new value.

<Story of={Stories.ControlledValue} />
