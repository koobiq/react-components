# Checkbox

<Status variant="experimental" />

A checkbox is used in forms when a user needs to select multiple values from a range of available options.

```tsx
import { Checkbox } from '@koobiq/react-components';
```

## Value

### Default value

Checkboxes are not checked by default. The `defaultSelected` prop can be used to set the default state.

<Story of={Stories.DefaultValue} />

### Controlled value

The `isSelected` prop can be used to make the checked state controlled.
The `onChange` event is fired when the user presses the checkbox, and receives the new value.

<Story of={Stories.ControlledValue} />

## Invalid

The `isInvalid` prop toggles the invalid state.

<Story of={Stories.Invalid} />

## Indeterminate

A checkbox input can only have two states in a form: checked or unchecked.

Visually, there are three states a checkbox can be in: checked, unchecked, or indeterminate.

The `isInvalid` prop toggles the indeterminate state.
