# Tabs

<Status variant="experimental" />

Tabs organize content into multiple sections and allow users to navigate between them.

```tsx
import { Tabs, Tab } from '@koobiq/react-components';
```

<Story of={Stories.Base} />

## Props

<Props />

## Subcomponents

- `Tabs`: The main component to display a tab list.
- `Tab`: The component to display a tab item. The children of this component will be displayed as the content of the tab.

## Disabled Item

Tabs supports marking items as disabled using the `disabledKeys` prop.
Each key in this list corresponds with the `key` prop passed to the `Tab` component.
